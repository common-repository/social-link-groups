Another best plugin for linking to your social media profiles in your website.

== Description ==

The Social Link Groups plugin adds a shortcode to your WordPress website allowing you to display icons linking to your social media profiles to your website. It's allowed to create a separated group of social link icon group to display it freely in any WordPress page, footer and header using shortcodes of each group. The Social Link Group plugin which lets you add icons for Facebook, Twitter, YouTube, Instagram, LinkedIn, Tumblr, Skype and many more. It's one of the best plugin for WooCommerce and eCommerce WordPress website for link social media profiles. With it, you can easily choose which profiles to link to your social media icons, as well as align them to the left, center, or right with stickey using shortcode.   

Features:-
1 - Normal and Sticky(left side / right side) icons view settings for a particular social link group.
2 - Manage the display position of each social icon.
3 - It's allowing admin to use a custom icon image for links as per website theme.
4 - Admin can manage to display a social link icon by its need.

Included Icons:-
- Facebook 
- Twitter
- YouTube
- Instagram
- LinkedIn
- Tumblr
- Skype
- Line
